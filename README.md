# EOS接龙竞拍区块链游戏
此游戏主要是以成语接龙为核心思想研发而成，用户可以通过点数挖矿的方式获得代币，同时根据其挖矿的点数进行接龙，如果用户的点数与之前其他用户的点数相同，则获得挖矿的代币奖励。
